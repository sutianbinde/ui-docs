    用于页面中展示重要的提示信息。

    # 为卡片中的图片绑定来源

    传入imgSrc 为卡片中的图片绑定来源，卡片可以声明一个类名来自定义一些样式,传入animation将启用简单动画,传入useful-frame来裁切并突出显示局部图片

    使用mark-shape属性定义要在图片上进行标记的形状，当不指定此值时，标记功能默认关闭。当图片标记功能开启时，通过mark-coordinate字段可以传入标记坐标值。通过mark-border设置标记描边样式

    <template>
  <div class="card-wrapper">
    <div
      class="card-box"
      v-for="(item,index) in cardData"
      :key="index">
      <meg-card
        class="card-item"
        :animation="true"
        :useful-frame="index===3 ? {'heightPixels': 93,'leftPixels': 152,'topPixels': 81,'widthPixels': 93} : {}"
        :mark-shape="index===4 ? 'rect' : ''"
        :mark-coordinate="index===4 ? {'heightPixels': 93,'leftPixels': 152,'topPixels': 81,'widthPixels': 93} : ''"
        :img-src="item.imgSrc">
      </meg-card>
    </div>
  </div>
</template>

    right-width 卡片右侧部分内容的宽度 string — 100px
    img-margin 卡片中图片的外边距 string — 5px
    img-border 卡片中图片的边框 string — ''
    show-angle 是否显示卡片周围的四个边角 boolean true/false false
    animation 是否启用动画（向上偏移2像素，0.5s） boolean true/false false
    useful-frame 图片有效数据的描述 object — {'heightPixels': 0,'leftPixels': 0,'topPixels': 0,'widthPixels': 0}
    context-download 是否支持右键下载原图 boolean — false
    img-title 图片区域的title提示 string — ''
    mark-shape 要在图片上标记的形状 string rect/line ''
    mark-coordinate 要标记的形状的坐标数据 object 框坐标值格式同useful-frame -
    mark-border 要标记的形状的线段样式 string - '2px solid red'
    enlarge 是否放大框 boolean - 默认会放大框（约1.5倍）
    width-scale 图像宽度缩放值 number - 0.8
    height-scale 图像高度缩放值 number - 0.8
    debug-host 已废弃，改为通过url传debug-host。本地环境图片的域 string - ''
    img-draggable 是否开启图片拖拽 boolean true/false true

    # Slot

    name 说明
    roi roi区域
    right 卡片右侧内容区
    bottom 卡片底部内容区，底部区域长度为图片区和右侧区长度之和
    custom 卡片自定义内容区，卡片为相对定位，所以自定义内容区可以绝对定位到卡片任意位置 ，可以有多个内容，比如左上角和右下角各一个图标，只需单独设置即可
    img-custom 图片自定义内容区，图片为相对定位，所以自定义内容区可以绝对定位到图片任意位置 ，可以有多个内容，比如左上角和右下角各一个图标，只需单独设置即可

    # Events

    事件名称 说明 回调参数
    click 卡片点击事件 若为img,则代表事件由图片发出，否则是卡片其余部分
    enter 鼠标进入卡片事件 若为img,则代表事件由图片发出，否则是卡片其余部分
    leave 鼠标离开卡片事件 若为img,则代表事件由图片发出，否则是卡片其余部分
    img-load 图片加载成功 图片对象
    img-error 图片加载失败 null
