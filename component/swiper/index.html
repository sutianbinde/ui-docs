    # Swiper 轮播

    利用 可视区渲染 实现的 支持数万数据量的高性能 轮播组件。

    WARNING

    • 如果出现切换之后稍 ui 视觉上 微抖动的情况，传入 item-key 即可解决
    • 如果组件在隐藏的情况下传入数据，则会出现显示宽度挤压的情况，需要在显示后再更新数据(比如：此组件用在弹窗中，改变弹窗为显示之后，再传入此组件的数据才能正常显示)，方案2：调用 resize()重新调整样式， 方案3：传入 width 参数固定宽度

    # 基础用法

    当前只支持横向轮播。

    使用 slides-per-view配置一屏中显示的切片数
    使用 loop配置是否循环
    使用 autoplay配置是否自动播放
    使用 pagination配置是否显示分页器
    使用 arrow配置左右箭头出现的方式

      <template>
  <div>
    <div
      class="wrap">
      <meg-swiper
        :data-list="swiperData"
        :slides-per-view="1"
        :loop="true"
        :autoplay="true"
        :pagination="true"
        :arrow="'hover'">
        <template v-slot="{data}">
          <div
            class="item"
            :style="{'background-color': data.color}">
            {{ data.name }}
          </div>
        </template>
      </meg-swiper>
    </div>
  </div>
</template>
<script>
export default {
  data() {
    return {
      swiperData: [],
    };
  },
  created() {
    for (let i = 0; i < 6; i += 1) {
      this.swiperData.push({
        name: `数据${i}`,
        color: this.getColor()
      });
    }
  },
  methods: {
    getColor(min = 150, max = 240) {
      return `rgba(${this.getOneColor(min, max)}, ${this.getOneColor(min, max)}, ${this.getOneColor(min, max)},1)`;
    },
    getOneColor(min, max) {
      const dis = max - min;
      return Math.floor(Math.random() * dis) + min;
    }
  }
};
</script>

  text-align: center;
  color: #193558;
}

      # 外部操作滚动

      当前只支持横向轮播。

      使用 default-index 配置默认移动到的索引
      使用 space-between 配置每个切片的间距
      使用 next 方法 后一页
      使用 prev 方法 前一页
      使用 slideToIndex 方法 滑动到指定item。

      <template>
  <div>
    <div
      class="wrap">
      <meg-swiper
        ref="swiper"
        :data-list="swiperData"
        :slides-per-view="5"
        :loop="true"
        :autoplay="false"
        :speed="300"
        :default-index="30"
        :space-between="10"
        @slide-change="slideChange">
        <template v-slot="{data}">
          <div
            class="item"
            :style="{'background-color': data.color}">
            {{ data.name }}
          </div>
        </template>
      </meg-swiper>
    </div>
    <div style="margin-top:10px">
      <meg-button @click="onPrev">
        上一页
      </meg-button>
      <meg-button @click="onNext">
        下一页
      </meg-button>
      <meg-input
        @change="onSlideTo"
        v-model="input1"
        style="width:100px; margin-left:10px"></meg-input>
      <meg-button @click="onSlideTo">
        跳转
      </meg-button>
    </div>
  </div>
</template>
<script>
export default {
  data() {
    return {
      swiperData: [],
      input1: 200,
    };
  },
  created() {
    for (let i = 0; i < 603; i += 1) {
      this.swiperData.push({
        name: `数据${i}`,
        color: this.getColor()
      });
    }
  },
  methods: {
    getColor(min = 150, max = 240) {
      return `rgba(${this.getOneColor(min, max)}, ${this.getOneColor(min, max)}, ${this.getOneColor(min, max)},1)`;
    },
    getOneColor(min, max) {
      const dis = max - min;
      return Math.floor(Math.random() * dis) + min;
    },
    onNext() {
      this.$refs.swiper.next();
    },
    onPrev() {
      this.$refs.swiper.prev();
    },
    onSlideTo() {
      this.$refs.swiper.slideToIndex(this.input1);
    },
    slideChange() { // 参数 nowIndex, nowView
      // console.log(nowIndex, nowView);
    }
  }
};
</script>

      # Attributes

      参数 说明 类型 可选值 默认值
      data-list 数据列表 Array — []
      width 自定义宽度(未传则以swipter 外层clientWidth 为宽) Number — -
      default-index 默认移动到的索引 Number — 0
      slides-per-view 一栏中显示的切片数 Number — 5
      space-between 每个切片的间距 Number — 0
      autoplay 是否自动切换 Boolean — false
      speed 切换速度 毫秒 Number — 300
      loop 是否循环 Boolean — false
      interval 自动切换间隔 毫秒 Number — 3000
      arrow 左右箭头 String always/hover/never never
      pagination 是否显示分页器 Boolean - false
      item-key 节点 key,作为 item 更新的依据，不传，则以 index 为 key，如果出现切换之后稍 ui 视觉上 微抖动的情况，传入 item-key 即可解决 String - -

      # Methods

      方法名称 说明 参数
      next 下一个 —
      prev 上一个 -
      slideToIndex 切换到某个 (index)对应item的 index 值
      slideToView 切换到某一屏 （view）视窗索引，既第几屏
      resize 刷新样式到和父级匹配的宽度 -

      # Scoped Slot

      Name Description
      - 自定义每个 item 的内容，参数为 { item }

      # Events

      事件名称 说明 回调参数
      init 初始化完成 —
      slide-change 初始化完成 nowIndex, nowView
