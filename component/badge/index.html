    # Badge 标记

    出现在按钮、图标旁的数字或状态标记。

    # 基础用法

    展示新消息数量。

    定义value属性，它接受Number或者String。。

    12
    3
    1
    2
    点我查看
    • 评论
      12
    • 回复
      3
    <meg-badge :value="12" class="item">
  <meg-button size="small">评论</meg-button>
</meg-badge>
<meg-badge :value="3" class="item">
  <meg-button size="small">回复</meg-button>
</meg-badge>
<meg-badge :value="1" class="item" type="primary">
  <meg-button size="small">评论</meg-button>
</meg-badge>
<meg-badge :value="2" class="item" type="warning">
  <meg-button size="small">回复</meg-button>
</meg-badge>

    点我查看<i class="meg-icon-caret-bottom meg-icon--right"></i>

    # 最大值

    可自定义最大值。

    由max属性定义，它接受一个Number，需要注意的是，只有当value为Number时，它才会生效。

    99+
    10+
    <meg-badge :value="200" :max="99" class="item">
  <meg-button size="small">评论</meg-button>
</meg-badge>
<meg-badge :value="100" :max="10" class="item">
  <meg-button size="small">回复</meg-button>
</meg-badge>
    show code

    # 自定义内容

    可以显示数字以外的文本内容。

    定义value为String类型是时可以用于显示自定义文本。

    new
    hot
    <meg-badge value="new" class="item">
  <meg-button size="small">评论</meg-button>
</meg-badge>
<meg-badge value="hot" class="item">
  <meg-button size="small">回复</meg-button>
</meg-badge>
    show code

    # 小圆点

    以圆点的形式标注需要关注的内容。

    除了数字外，设置is-dot属性，它接受一个Boolean。

    数据查询 
    <meg-badge is-dot class="item">数据查询</meg-badge>
<meg-badge is-dot class="item">
  <meg-button class="share-button" icon="meg-icon-share" type="primary"></meg-button>
</meg-badge>
    show code

    # Attributes

    参数 说明 类型 可选值 默认值
    value 显示值 string, number — —
    max 最大值，超过最大值会显示 '{max}+'，要求 value 是 Number 类型 number — —
    is-dot 小圆点 boolean — false
    hidden 隐藏 badge boolean — false
    type 类型 string primary / success / warning / danger / info —
