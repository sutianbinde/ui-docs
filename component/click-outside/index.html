    # ClickOutside 指令

    外部点击事件监听指令。

    # 基础用法

    使用 v-click-outside 指令，可以在点击指定组件外的任意区点击触发回调。

    父元素

    该区域为指令绑绑定区域

    <template>
  <div class="block">
    <div class="click-area">
      <p>父元素</p>
      <div
        class="click-area-bind"
        v-click-outside="onClickOutside">
        <p>该区域为指令绑绑定区域</p>
      </div>
    </div>
  </div>
</template>
<script>
export default {
  data() {
    return {
    };
  },
  methods: {
    onClickOutside() {
      this.$message({
        message: '点击外部了',
        type: 'success'
      });
    }
  }
};
</script>
<style lang="css" scoped>
.click-area, .click-area-bind{
  display: flex;
  justify-content: center;
  align-items: center;
  border-radius: 5px;
}
.click-area {
  flex-direction: column;
  background-color: #d3dce6;
  width: 600px;
  height: 300px;
}
.click-area-bind {
  background-color: #99a9bf;
  width: 400px;
  height: 100px;
}
</style>
