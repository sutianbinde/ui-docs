# ImgSelect 图片选择组件

用于从本地选择图片的基础组件

注意

图片选择组件仅用于选择图片文件，如有其它文件选择需求，请使用其它组件

# 用法

# 基础用法

图片选择具有默认的UI，如果不自定义slot:default则使用默认UI

提示

使用默认UI时，可以为组件单独提供一个slot:trigger用于自定义其它触发操作，请确保一定在默认UI下使用该slot

提示

可以通过组件触发选择，也可以通过方法触发选择，不同触发方式在事件中回调参数的selectType值不同

提示

组件默认支持拖拽选择图片

请选择图片
upload from history 
<template>
  <div class="demo">
    <meg-img-select
      ref="imgSelect"
      style="width:300px;height:300px"
      tip="请选择图片"
      :img-url="imgSrc"
      :multiple="false"
      @file-changed="onImgChanged"
      @invalidate="onInvalidate">
      <template v-slot:trigger>
        <div
          class="upload-slot"
          @click="onClickFromTrigger">
          upload from history
        </div>
      </template>
    </meg-img-select>
    <div>
      <meg-button
        type="primary"
        @click="onClickSelect">
        选择
      </meg-button>
      <meg-button
        type="primary"
        @click="onClickRemove">
        删除
      </meg-button>
    </div>
  </div>
</template>
<script>
export default {
  data() {
    return {
      imgSrc: '',
      selectFiles: []
    };
  },
  methods: {
    onImgChanged(files, selectType) {
      this.$message({
        type: 'info',
        message: selectType
      });
    },
    onClickSelect() {
      this.$refs.imgSelect.selectImage();
    },
    onClickRemove() {
      this.$refs.imgSelect.removeImage();
    },
    onClickFromTrigger() {
      this.$message('trigger');
    },
    onInvalidate(type) {
      this.$message({
        type: 'warning',
        message: type
      });
    },
    onFileChanged(fileList) {
      this.selectFiles = fileList;
    }
  }
};
</script>

# 自定义UI以及图片格式

组件可以自定义UI，并自动绑定click事件，点击后通过事件回调获取选择的图片，选择图片结果支持url和base64两种格式

提示

自定义UI默认会支持拖拽选择图片

选择结果： 
<template>
  <div class="demo">
    <div>
      <meg-img-select
        file-type="base64"
        :img-url="imgSrc"
        :limit="2"
        :multiple="true"
        @file-changed="onFileChanged"
        @invalidate="onInvalidate">
        <template v-slot:default>
          <meg-button>图片选择</meg-button>
        </template>
      </meg-img-select>
      <div style="margin:20px 0">
        选择结果：
      </div>
      <div>
        <div
          class="img-info"
          v-for="(file,index) in selectFiles"
          :key="index">
          {{ `base64: ${file}` }}
        </div>
      </div>
    </div>
  </div>
</template>
<script>
export default {
  data() {
    return {
      imgSrc: '',
      selectFiles: []
    };
  },
  methods: {
    onClickSelect() {
      this.$refs.imgSelect.selectImage();
    },
    onClickRemove() {
      this.$refs.imgSelect.removeImage();
    },
    onClickFromTrigger() {
      this.$message('trigger');
    },
    onInvalidate(type) {
      this.$message({
        type: 'warning',
        message: type
      });
    },
    onFileChanged(fileList) {
      this.selectFiles = fileList;
    }
  }
};
</script>

# 组件参数

# Prop

参数 说明 类型 可选值 默认值
placeholder-image 站位图图片 string - -
name 表单对应的字段名 string - ‘’
tip 下部提示语 string - ‘’
file-type 返回图片的文件形式 string 'file' 'base64' 'file'
multiple 是否允许多选 boolean — true
limit 文件选择数量，不设置则不限制 number — undefined
max-size 选择文件的大小上限（单位：M） number — 10
accept-img-type 可接受的文件类型 array - []

# Event

事件名称 说明 回调参数
file-changed 选择文件变化时触发 files:文件列表 selectType:原生触发为‘native’，通过方法触发时为‘method’
invalidate 选择文件验证不通过时触发 reason：数量超出是为‘limit’ 大小超出时为‘size’ 类型不合适时为‘type’
remove 选择文件删除时触发 files:删除的文件列表

# Methods

方法名 说明 参数
removeImage() 清除选择图片 -
selectImage() 触发选择图片 -

# Slot

name 说明
default 默认插槽
trigger 其它触发方式插槽，default存在时有效
