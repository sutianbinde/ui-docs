# 开发指南

# 安装

确保可访问私服源：

# 新私服
registry=http://10.199.1.17:7001/

# 旧私服
registry=http://10.199.1.20:8081/repository/npmmegvii/

npm install megvii-ui@beta --save

# 引入

在 main.js 中写入以下内容：

import Vue from 'vue'
import MegviiUI from 'megvii-ui'
import 'megvii-ui/dist/megvii-ui.css'
import App from './App.vue'

Vue.use(MegviiUI)

new Vue({
  el: '#app',
  render: h => h(App)
})

以上代码便完成了 MegviiUI 的引入。需要注意的是，样式文件需要单独引入。

# 全局配置

在引入 MegviiUI 时，可以传入一个全局配置对象。该对象目前支持 size 与 zIndex 字段。size 用于改变组件的默认尺寸，zIndex 设置弹框的初始 z-index（默认值：2000）。按照引入 MegviiUI 的方式，具体操作如下：

import Vue from 'vue';
import MegviiUI from 'megvii-ui'

Vue.use(MegviiUI, { size: 'small', zIndex: 3000 });

# 国际化

megvii-ui 组件内部默认使用中文，若希望使用其他语言，则需要进行多语言设置。以英文为例，在 main.js 中：

import Vue from 'vue';
import MegviiUI from 'megvii-ui'
import locale from 'megvii-ui/lib/utils/locale/lang/en'

Vue.use(MegviiUI, { locale });

目前 MegviiUI 内置了以下语言：

  • 简体中文（zh-CN）
  • 英语（en）

# 自定义主题

通过覆盖内置样式变量可以实现自定义主题，注意需要重新覆盖$--assets-path，在项目里新建custom.scss，然后引入：

// $--color-primary: red;
$--assets-path: '~megvii-ui/lib/style/assets';
@import '~megvii-ui/lib/index.scss';
