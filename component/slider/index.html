    # Slider 滑块

    通过拖动滑块在一个固定区间内进行选择

    # 基础用法

    在拖动滑块时，显示当前值

    通过设置绑定值自定义滑块的初始值

    默认
    自定义初始值
    隐藏 Tooltip
    格式化 Tooltip
    禁用 
    <template>
  <div>
    <div class="block">
      <span class="demonstration">默认</span>
      <meg-slider v-model="value1"></meg-slider>
    </div>
    <div class="block">
      <span class="demonstration">自定义初始值</span>
      <meg-slider v-model="value2"></meg-slider>
    </div>
    <div class="block">
      <span class="demonstration">隐藏 Tooltip</span>
      <meg-slider
        v-model="value3"
        :show-tooltip="false"></meg-slider>
    </div>
    <div class="block">
      <span class="demonstration">格式化 Tooltip</span>
      <meg-slider
        v-model="value4"
        :format-tooltip="formatTooltip"></meg-slider>
    </div>
    <div class="block">
      <span class="demonstration">禁用</span>
      <meg-slider
        v-model="value5"
        disabled></meg-slider>
    </div>
  </div>
</template>

<script>
export default {
  data() {
    return {
      value1: 0,
      value2: 50,
      value3: 36,
      value4: 48,
      value5: 42
    };
  },
  methods: {
    formatTooltip(val) {
      return val / 100;
    }
  }
};
</script>

    # 离散值

    选项可以是离散的

    改变step的值可以改变步长，通过设置show-stops属性可以显示间断点

    不显示间断点
    显示间断点 
    <template>
  <div>
    <div class="block">
      <span class="demonstration">不显示间断点</span>
      <meg-slider
        v-model="value6"
        :step="10">
      </meg-slider>
    </div>
    <div class="block">
      <span class="demonstration">显示间断点</span>
      <meg-slider
        v-model="value7"
        :step="10"
        show-stops>
      </meg-slider>
    </div>
  </div>
</template>

<script>
export default {
  data() {
    return {
      value6: 0,
      value7: 0
    };
  }
};
</script>
    show code

    # 带有输入框

    通过输入框设置精确数值

    设置show-input属性会在右侧显示一个输入框

    <template>
  <div class="block">
    <meg-slider
      v-model="value8"
      show-input>
    </meg-slider>
  </div>
</template>

<script>
export default {
  data() {
    return {
      value8: 0
    };
  }
};
</script>
    show code

    # 范围选择

    支持选择某一数值范围

    设置range即可开启范围选择，此时绑定值是一个数组，其元素分别为最小边界值和最大边界值

    <template>
  <div class="block">
    <meg-slider
      v-model="value9"
      range
      show-stops
      :max="10">
    </meg-slider>
  </div>
</template>

<script>
export default {
  data() {
    return {
      value9: [4, 8]
    };
  }
};
</script>

    # 竖向模式

    设置vertical可使 Slider 变成竖向模式，此时必须设置高度height属性

    <template>
  <div class="block">
    <meg-slider
      v-model="value10"
      vertical
      height="200px">
    </meg-slider>
  </div>
</template>

<script>
export default {
  data() {
    return {
      value10: 0
    };
  }
};
</script>

    # Attributes

    参数 说明 类型 可选值 默认值
    value / v-model 绑定值 number — 0
    min 最小值 number — 0
    max 最大值 number — 100
    disabled 是否禁用 boolean — false
    step 步长 number — 1
    show-input 是否显示输入框，仅在非范围选择时有效 boolean — false
    show-input-controls 在显示输入框的情况下，是否显示输入框的控制按钮 boolean — true
    input-size 输入框的尺寸 string large / medium / small / mini small
    show-stops 是否显示间断点 boolean — false
    show-tooltip 是否显示 tooltip boolean — true
    format-tooltip 格式化 tooltip message function(value) — —
    range 是否为范围选择 boolean — false
    vertical 是否竖向模式 boolean — false
    height Slider 高度，竖向模式时必填 string — —
    label 屏幕阅读器标签 string — —
    debounce 输入时的去抖延迟，毫秒，仅在show-input等于true时有效 number — 300
    tooltip-class tooltip 的自定义类名 string — —
