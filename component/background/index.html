    # Background 内容区背景

    用于页面中展示重要的提示信息。

    # 自定义指令v-background

    页面中的非浮层元素，不会自动消失。

    使用v-background来控制时候显示默认背景、图标、提示文字。

    提示

    • 原icon_1, icon_2, icon_4被替换为了默认的common_empty
    • 内置四种空状态图标及对应的icon值：通用空状态(common_empty)、人机非空状态(man_motor_empty)、人脸(face)、人体(body)

    # 通用空

    <template>
  <div>
    <div
      class="card-wrapper"
      v-background="option">
    </div>
  </div>
</template>

<script>
export default {
  data() {
    return {
      option: {
        icon: 'common_empty',
        text: '暂无数据'
      },
    };
  }
};
</script>

<style lang="scss" scoped>

.card-wrapper{
  width: 100%;
  margin-top:20px;
  height: 180px;
}
</style>

    # 切换背景

    使用v-background 为 false 来隐藏背景。

    <template>
  <div>
    <meg-button @click="changebg">
      点我切换背景
    </meg-button>
    <meg-button @click="hidebg">
      点我隐藏背景
    </meg-button>
    <div
      class="card-wrapper"
      v-background="visible">
    </div>
  </div>
</template>


<script>
export default {
  data() {
    return {
      visible: true,
    };
  },
  methods: {
    changebg() {
      this.visible = this.visible === true ? { icon: 'icon_4', text: '暂无结果' } : true;
    },
    hidebg() {
      this.visible = false;
    }
  }
};
</script>

<style lang="scss" scoped>

.card-wrapper {
  width: 100%;
  margin-top:20px;
  height: 180px;
}
</style>

    # 自定义显示图标

    使用imageSrc显示自定义的图标

    <template>
  <div>
    <div
      class="card-wrapper"
      v-background="option">
    </div>
  </div>
</template>

<script>
export default {
  data() {
    return {
      option: {
        imageSrc: 'https://ss1.bdstatic.com/70cFuXSh_Q1YnxGkpoWK1HF6hhy/it/u=1717303849,2480626072&fm=200&gp=0.jpg'
      },
    };
  }
};
</script>

<style lang="scss" scoped>

.card-wrapper{
  width: 100%;
  margin-top:20px;
  height: 180px;
}
</style>

    # Attributes

    参数 说明 类型 可选值 默认值
    text 提示文字 string - -
    image 背景图片 string bg_1 bg_1
    noImage 不显示背景 boolean - false
    imageSrc 自定义图标 string - -
    icon 内置图标 string face, body, common_empty, man_motor_empty common_empty
    margin 提示文字的周围间距 string - -
