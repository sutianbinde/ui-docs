    # DoubleCard 双人像卡片

    自定义的风格统一的双人像卡片组件

    # 为卡片中的图片绑定来源

    传入leftImgSrc，rightImgSrc， 为卡片中的图片绑定来源， 卡片可以声明一个类名来自定义一些样式， 当使用single参数时，将只渲染leftImgSrc为单头像模式，rightImgSrc无效, 传入animation将启用简单动画， 传入 width 改变图片占比

    传入left-useful-frame和right-useful-frame来裁切并突出显示局部图片

    使用left/right-mark-shape属性定义要在图片上进行标记的形状，当不指定此值时，标记功能默认关闭。当图片标记功能开启时，通过left/right-mark-coordinate字段可以传入标记坐标值。通过left/right-mark-border设置标记描边样式

    <template>
  <div class="card-wrapper">
    <div
      class="card-box"
      v-for="(item,index) in cardData"
      :key="index">
      <meg-double-card
        class="card-item"
        width="50%"
        :animation="true"
        :single="index!==0"
        :left-useful-frame="index===2 ? {'heightPixels': 93,'leftPixels': 152,'topPixels': 81,'widthPixels': 93} : {}"
        :right-useful-frame="index===2 ? {'heightPixels': 93,'leftPixels': 152,'topPixels': 81,'widthPixels': 93} : {}"
        :left-img-src="item.leftImgSrc"
        :right-img-src="item.rightImgSrc"
        :left-mark-shape="index===3 ? 'rect' : ''"
        :left-mark-coordinate="index===3 ? {'heightPixels': 93,'leftPixels': 152,'topPixels': 81,'widthPixels': 93} : ''">
      </meg-double-card>
    </div>
  </div>
</template>

    # Attributes

    参数 说明 类型 可选值 默认值
    left-img-src 卡片中左侧图片地址 string — —
    right-img-src 卡片中右侧图片地址 string — —
    score 卡片中的比对分数 number — -1
    width 卡片中单张图片占卡片宽度的百分比,因为对称关系，该百分比不会超过50%，剩余部分自动分给侧边留白 string — '37.5%'
    animation 是否启用动画（向上偏移2像素，0.5s） boolean true/false false
    left-useful-frame 左侧图片有效数据的描述 object — {'heightPixels': 0,'leftPixels': 0,'topPixels': 0,'widthPixels': 0}
    right-useful-frame 右侧图片有效数据的描述 object — {'heightPixels': 0,'leftPixels': 0,'topPixels': 0,'widthPixels': 0}
    context-download 是否支持右键下载原图 boolean — false
    left-mark-shape 左图 要在图片上标记的形状 string rect/line ''
    left-mark-coordinate 左图 要标记的形状的坐标数据 object 框坐标值格式同useful-frame -
    left-mark-border 左图 要标记的形状的线段样式 string - '2px solid red'
    right-mark-shape 右图 要在图片上标记的形状 string rect/line ''
    right-mark-coordinate 右图 要标记的形状的坐标数据 object 框坐标值格式同useful-frame -
    right-mark-border 右图 要标记的形状的线段样式 string - '2px solid red'
    left-enlarge 左图 是否放大框 boolean - 默认会放大框（约1.5倍）
    right-enlarge 右图 是否放大框 boolean - 默认会放大框（约1.5倍）
    left-width-scale 左图宽度缩放值 number - 0.8
    right-width-scale 右图宽度缩放值 number - 0.8
    left-height-scale 左图高度缩放值 number - 0.8
    right-height-scale 右图高度缩放值 number - 0.8
    debug-host 已废弃，改为通过url传debug-host。本地环境图片的域 string - ''
    single 是否是单张图片(单张图片时，只有左图相关属性生效) boolean - false
    left-img-draggable 左侧图片是否支持拖拽 boolean true/false true
    right-img-draggable 右侧图片是否支持拖拽 boolean true/false true

    # Slot

    name 说明
    header 卡片顶部内容区
    bottom 卡片底部内容区
    custom 卡片自定义内容区，卡片为相对定位，所以自定义内容区可以绝对定位到卡片任意位置 ，可以有多个内容，比如左上角和右下角各一个图标，只需单独设置即可
    left-custom 左侧图片自定义内容区，图片为相对定位，所以自定义内容区可以绝对定位到图片任意位置 ，可以有多个内容，比如左上角和右下角各一个图标，只需单独设置即可
    right-custom 右侧图片自定义内容区，图片为相对定位，所以自定义内容区可以绝对定位到图片任意位置 ，可以有多个内容，比如左上角和右下角各一个图标，只需单独设置即可

    # Events

    事件名称 说明 回调参数
    click 卡片点击事件 若为left,则代表事件由左图片发出，若为right,则代表事件由右图片发出，否则是卡片其余部分
    enter 鼠标进入卡片事件 若为left,则代表事件由左图片发出，若为right,则代表事件由右图片发出，否则是卡片其余部分
    leave 鼠标离开卡片事件 若为left,则代表事件由左图片发出，若为right,则代表事件由右图片发出，否则是卡片其余部分
    img-error 图片加载错误事件 若为left,则代表事件由左图片发出，若为right,则代表事件由右图片发出
