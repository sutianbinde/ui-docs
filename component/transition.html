    # 内置过渡动画

    megvii 内应用在部分组件的过渡动画，你也可以直接使用。在使用之前请阅读 transition 组件文档 。

    # fade 淡入淡出

    提供 meg-fade-in-linear 和 meg-fade-in 两种效果。

    .meg-fade-in-linear
    .meg-fade-in
    <template>
  <div>
    <meg-button @click="show = !show">Click Me</meg-button>
    <div style="display: flex; margin-top: 20px; height: 100px;">
      <transition name="meg-fade-in-linear">
        <div v-show="show" class="transition-box">.meg-fade-in-linear</div>
      </transition>
      <transition name="meg-fade-in">
        <div v-show="show" class="transition-box">.meg-fade-in</div>
      </transition>
    </div>
  </div>
</template>

<style scoped>
  .transition-box {
    margin-bottom: 10px;
    width: 200px;
    height: 100px;
    border-radius: 4px;
    background-color: #99a9bf;
    text-align: center;
    color: #fff;
    padding: 40px 20px;
    box-sizing: border-box;
    margin-right: 20px;
  }
</style>

    # zoom 缩放

    提供 meg-zoom-in-center，meg-zoom-in-top 和 meg-zoom-in-bottom 三种效果。

    .meg-zoom-in-center
    .meg-zoom-in-top
    .meg-zoom-in-bottom
    <template>
  <div>
    <meg-button @click="show2 = !show2">Click Me</meg-button>
    <div style="display: flex; margin-top: 20px; height: 100px;">
      <transition name="meg-zoom-in-center">
        <div v-show="show2" class="transition-box">.meg-zoom-in-center</div>
      </transition>
      <transition name="meg-zoom-in-top">
        <div v-show="show2" class="transition-box">.meg-zoom-in-top</div>
      </transition>
      <transition name="meg-zoom-in-bottom">
        <div v-show="show2" class="transition-box">.meg-zoom-in-bottom</div>
      </transition>
    </div>
  </div>
</template>

  .transition-box {

    background-color: #99a9bf;
