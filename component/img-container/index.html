    # ImgContainer 图片容器组件

    自定义的图片容器组件。

    # 基础用法

    封装、简化系统中图片容器的处理，默认使图片在容器中按照宽高的最大值充满容器且不发生形变。

    使用tile属性来定义图片在容器中是否平铺的开关,:img-src属性来绑定需要显示图片的url，指定vertical-align和text-align属性可以控制图片在容器内的对齐方式。

    <template>
  <div>
    <div
      class="block"
      style="display: flex">
      <transition
        name="meg-zoom-in-center">
        <div
          class="block-item"
          v-show="imgVisible1">
          <meg-img-container
            :img-src="imgSrc1">
          </meg-img-container>
        </div>
      </transition>
      <div
        class="block-item">
        <meg-img-container
          :img-src="imgSrc2">
        </meg-img-container>
      </div>
      <div
        class="block-item">
        <meg-img-container
          vertical-align="top"
          :img-src="imgSrc2">
        </meg-img-container>
      </div>
    </div>
    <div class="block">
      <meg-button
        @click="imgVisible1=!imgVisible1">
        显示隐藏图片
      </meg-button>
      <meg-button
        @click="changeImg">
        改变图片
      </meg-button>
    </div>
  </div>
</template>
<script>
const img1 = '/ui-docs/demo.jpeg';
const img2 = '/ui-docs/demo.jpeg';

export default {
  data() {
    return {
      imgSrc1: img1,
      imgSrc2: img1,
      imgVisible1: false,
    };
  },
  methods: {
    changeImg() {
      this.imgSrc2 = this.imgSrc2 === img1 ? img2 : img1;
    }
  }
};
</script>
<style lang="scss" scoped>
  .block {
    margin-bottom: 10px;
    .block-item{
      display: inline-block;
      width: 200px;
      height: 200px;
      margin: 0 5px;
      border: 1px solid #dcdfe6;
    }
  }
</style>

    # 自动裁剪图片显示和右键下载原图

    组件中集成了根据传入的裁剪框坐标自动裁图展示和右键下载原图的功能。

    根据业务需求，组件中默认集成了自动裁剪图片显示和右键下载原图的功能，可以通过useful-frame字段传入要在原图上裁剪的图片的坐标。右键下载功能可以通过context-download开关进行开启。 可通过download-menu-text 和 download-file-name 配置右键菜单显示的文字 及 下载文件的文件名

    <template>
  <div
    class="block-item">
    <meg-img-container
      :img-src="imgSrc1"
      :useful-frame="imgDesc"
      :context-download="true">
    </meg-img-container>
    <meg-img-container
      :img-src="imgSrc1"
      :useful-frame="imgDesc"
      download-menu-text="下载全景图"
      download-file-name="bigimage"
      :context-download="true">
    </meg-img-container>
  </div>
</template>

<script>

const img1 = '/ui-docs/demo.jpeg';
export default {
  data() {
    return {
      imgSrc1: img1,
      imgDesc: {
        heightPixels: 99,
        leftPixels: 144,
        topPixels: 71,
        widthPixels: 97,
      }
    };
  },
};
</script>

<style lang="scss" scoped>
 .block-item{
   display: flex;
   div{
     width: 200px;
     height: 200px;
     margin: 0 10px;
   }
 }
</style>

    # 图片加载错误

    当填入的图片加载错误时，显示一张默认错误提示图片。

    使用error-img属性定义是否开启图片加载错误显示默认图片的功能，默认开启。当默认错误图片功能开启时，通过error-img-src字段可以手动指定加载错误时显示的默认图。

    <template>
  <div class="container">
    <div class="img-box">
      <meg-img-container
        :img-src="RandomString"></meg-img-container>
    </div>
    <div class="img-box">
      <meg-img-container
        :img-src="RandomString"
        :error-img="false"></meg-img-container>
    </div>
    <div class="img-box">
      <meg-img-container
        :img-src="RandomString"
        :error-img-src="errorImg"></meg-img-container>
    </div>
  </div>
</template>

<script>
import errorImg from './images/error.png';

export default {
  data() {
    return {
      errorImg,
      RandomString: '',
    };
  },
};
</script>

<style lang="scss" scoped>
  .container{
    display: flex;
  }
  .img-box{
    width: 200px;
    border: 1px solid #dcdfe6;
    margin: 0 10px;
  }
</style>

    # 图片标记

    针对图片的标记需求封装，目前只支持在图片上画框操作，画线或者其他操作需要明确后端返回的数据结构后增加处理逻辑。

    使用mark-shape属性定义要在图片上进行标记的形状，当不指定此值时，标记功能默认关闭。当图片标记功能开启时，通过mark-coordinate字段可以传入标记坐标值。通过mark-border设置标记描边样式，默认为2px solid red，当传入false时不显示边框。通过enlarge设置是否按预设比例放大框。

    <template>
  <div>
    <div
      class="block"
      style="display: flex">
      <div
        class="block-item">
        <meg-img-container
          :img-src="imgSrc1"
          :mark-coordinate="imgDesc"
          mark-shape="rect">
        </meg-img-container>
      </div>
      <div
        class="block-item">
        <meg-img-container
          vertical-align="top"
          :img-src="imgSrc1"
          :mark-coordinate="imgDesc"
          :enlarge="false"
          mark-shape="rect">
        </meg-img-container>
      </div>
    </div>
    <div class="block">
    </div>
  </div>
</template>
<script>
const img1 = '/ui-docs/demo.jpeg';

export default {
  data() {
    return {
      imgSrc1: img1,
      imgDesc: {
        heightPixels: 95,
        leftPixels: 150,
        topPixels: 80,
        widthPixels: 95
      }
    };
  },
  methods: {
  }
};
</script>
<style lang="scss" scoped>
  .block {
    margin-bottom: 10px;
    .block-item{
      display: inline-block;
      width: 200px;
      height: 200px;
      margin: 0 5px;
      border: 1px solid #dcdfe6;
    }
  }
</style>

    # Attributes

    参数 说明 类型 可选值 默认值
    tile 图片是否在容器内平铺 boolean true/false false
    vertical-align 图片在容器中的垂直对齐方式 string top/middle/bottom middle
    text-align 图片在容器中的水平对齐方式 string left/center/right center
    img-src 容器中图片的Src string — —
    useful-frame 图片有效数据的描述 object — {'heightPixels': 0,'leftPixels': 0,'topPixels': 0,'widthPixels': 0}
    aspect-ratio 手动制定容器长宽比(如果浏览器不支持 object-fit 属性，可以使用此属性修正) number — 0
    context-download 是否支持右键下载原图 boolean — false
    error-img 当指定的图片地址图片不存在时是否显示默认图 boolean true/false true
    error-img-src 传入自定义的默认错误图片 string — -
    download-menu-text 右键下载提示文字 string — 下载原图
    download-file-name 下载文件的文件名 string — 下载原图
    mark-shape 要在图片上标记的形状 string rect/line ''
    mark-coordinate 要标记的形状的坐标数据 object 框坐标值格式同useful-frame -
    mark-border 要标记的形状的线段样式 string/boolean - '2px solid red'
    enlarge 是否放大框 boolean/object - 默认宽高缩放值为0.8，可以传入对象指定宽高的缩放值
    widthScale 宽度缩放值 number 0.8 图像宽度缩放值，配合useful-frame来使用
    heightScale 高度缩放值 number 0.8 图像高度缩放值，配合useful-frame来使用
    debugHost 已废弃，改为在url上传debug-host string 图片地址不带域时，在本地开发图片无法显示，可以通过这个参数指定一个域，默认为当前域
    draggable 是否开启拖拽 boolean true/false true
