    # InputNumber 计数器

    仅允许输入标准的数字值，可定义范围

    # 基础用法

    要使用它，只需要在meg-input-number元素中使用v-model绑定变量即可，变量的初始值即为默认值。

    <template>
  <meg-input-number
    v-model="num1"
    @change="handleChange"
    :min="1"
    :max="10"
    label="描述文字"></meg-input-number>
</template>
<script>
export default {
  data() {
    return {
      num1: 1
    };
  },
  methods: {
    handleChange(value) {
      console.log(value);
    }
  }
};
</script>
    show code

    # 禁用状态

    disabled属性接受一个Boolean，设置为true即可禁用整个组件，如果你只需要控制数值在某一范围内，可以设置min属性和max属性，不设置min和max时，最小值为 0。

    <template>
  <meg-input-number
    v-model="num2"
    :disabled="true"></meg-input-number>
</template>
<script>
export default {
  data() {
    return {
      num2: 1
    };
  }
};
</script>
    show code

    # 步数

    允许定义递增递减的步数控制

    设置step属性可以控制步长，接受一个Number。

    <template>
  <meg-input-number
    v-model="num3"
    :step="2"></meg-input-number>
</template>
<script>
export default {
  data() {
    return {
      num3: 5
    };
  }
};
</script>
    show code

    # 精度

    设置 precision 属性可以控制数值精度，接收一个 Number。

    <template>
  <meg-input-number
    v-model="num9"
    :precision="2"
    :step="0.1"
    :max="10"></meg-input-number>
</template>
<script>
export default {
  data() {
    return {
      num9: 1
    };
  }
};
</script>
    show code

    TIP

    precision 的值必须是一个非负整数，并且不能小于 step 的小数位数。

    # 尺寸

    额外提供了 medium、small、mini 三种尺寸的数字输入框

    <template>
  <div>
    <meg-input-number v-model="num4"></meg-input-number>
    <meg-input-number
      size="medium"
      v-model="num5"></meg-input-number>
    <meg-input-number
      size="small"
      v-model="num6"></meg-input-number>
    <meg-input-number
      size="mini"
      v-model="num7"></meg-input-number>
  </div>
</template>
<script>
export default {
  data() {
    return {
      num4: 1,
      num5: 1,
      num6: 1,
      num7: 1
    };
  }
};
</script>
    show code

    # Attributes

    参数 说明 类型 可选值 默认值
    value / v-model 绑定值 number — 0
    min 设置计数器允许的最小值 number — -Infinity
    max 设置计数器允许的最大值 number — Infinity
    step 计数器步长 number — 1
    precision 数值精度 number — —
    size 计数器尺寸 string large, small —
    disabled 是否禁用计数器 boolean — false
    controls 是否使用控制按钮 boolean — true
    controls-position 控制按钮位置 string right -
    name 原生属性 string — —
    label 输入框关联的label文字 string — —
    placeholder 输入框默认 placeholder string - -
    caret-style 加减图标使用caret风格 boolean - false

    # Events

    事件名称 说明 回调参数
    change 绑定值被改变时触发 最后变更的值
    blur 在组件 Input 失去焦点时触发 (event: Event)
    focus 在组件 Input 获得焦点时触发 (event: Event)
