    # TreeSelect 树选择器

    WARNING

    • 该组件是基础tree控件功能的扩充，基本用法与基础树相同

    # 基本用法

    通过getSelectedKeys方法可以获取到选项框中的选中项的key，通过getSelectedNodes方法可以获取到选项框中选中项的节点信息。

    # 基础参数

    和树几乎完全一致的基础参数

    meg-treeselect 透传了 meg-tree 所有的 参数 方法 props 事件 Scoped Slot 点击链接查看

    # 参数

    参数 说明 类型 可选值 默认值
    data 展示数据 array — —
    standard-tree-data 是否是标准的树结构数据(带层级关系的数据，非平铺数组)（此参数未来会废弃，一般不使用此参数) boolean — true
    props { children: 'xxx', label: 'xxx' }指定子节点 和 label 对应的字段 Object — { children: 'children', label: 'label' }
    check-strictly 在显示复选框的情况下，是否严格的遵循父子不互相关联的做法，默认为 false boolean — false
    expand-on-click-node 是否在点击节点的时候展开或者收缩节点， 默认值为 true，如果为 false，则只有点箭头图标的时候才会展开或者收缩节点 boolean — true
    default-checked-keys 默认勾选的节点的 key 的数组 array — —
    show-checkbox 节点是否可被选择 boolean — false
    highlight-current 是否高亮当前选中节点，默认值是 false boolean — false
    draggable 是否开启拖拽节点功能(此为开启树内部节点之间的拖拽，如果只设置某些节点是否可拖拽【向外拖拽】，请用can-drag-fn，并确保此参数为 false) boolean — false
    dropable 是否整个树的节点可以接受外来拖入 boolean — false
    default-checked-all 是否默认选中全部 boolean — false
    empty-text 空数据时显示的文本内容 String — 暂无数据
    render-all 是否全量渲染树 boolean — false
    content-height 将树渲染成固定高度（对于开始隐藏的情况很有用） Number — -
    can-drag-fn(data) 根据返回值判断某个节点是否可拖拽的方法【注意：此方法和draggable冲突，请勿同时设置draggable为 true】，拖拽出去以后，可通过e.dataTransfer.getData('draggingData')可获取到拖拽的节点的源数据(已转化为 String 类型)) Function(data) { return boolean } 参数为每个节点数据，返回true表示某个可拖动 — -
    pretreatment-node-data-fn(item) 对每个节点数据进行预处理， item 表示内部扁平化数据时遍历到的每个节点源数据的引用，直接修改，修改后，再进行后续处理，所以，请谨慎修改 Function — -
    node-key 作为 id 的替代项，将会以传入的 key 名称作为 node id 的映射 String — "id"
    default-expanded-keys 默认展开的节点的 key 的数组, 传入非空数组，将会只展开传入项及父级,出于性能考虑，只接收长度不大于30的数组 Array — []
    default-expand-all 是否展开全部(会被default-expanded-keys 覆盖) boolean — true
    item-class-fn(node) 根据返回值为节点添加独立的 class Function — -
    check-on-click-node 是否在点击节点的时候选中节点，默认值为 false，即只有在点击复选框时才会选中节点。 boolean — false
    filter-node-method(value, node) 对树节点进行筛选时执行的方法，返回 true 表示这个节点可以显示，返回 false 则表示这个节点会被隐藏 Function — -
    search-placeholder 检索框的 placeholder String — "请输入关键字进行检索"
    record-dynamic-select 显示选中的统计数据 boolean — false
    show-dept-icon 显示机构或区域图标（需提前在传入数据中添加 flag: 'dept'/'place' 表示是机构/区域） boolean — false
    show-camera-icon 显示机构图标（需提前在传入数据中添加 flag: 'camera' 表示是相机） boolean — false
    selected-count-text '已选 {count} 个' 的替代文字，组件内部会把 {count} 替换成选中的个数 String — '已选 {count} 个'
    placeholder-text "请选择相机" 的替代文字， String — "请选择相机"
    all-selected-text "全部相机" 的替代文字， String — "全部相机"
    show-count 是否显示统计数字 已选xxx个 boolean — true
    count-right-bottom 统计数字显示的位置 默认右上角，设置此为 true 则显示到右下角 boolean — false
    show-all-checked 是否在选中列表中显示所有选中项, 如果是 false，则只显示叶子节点（标记有noStatistics的项也不会显示） boolean — false
    disabled 是否禁用 boolean — false
    hide-popover-when-checked 当点击节点的时候 隐藏浮层 boolean — false
    single-check 是否为单选模式 boolean — false
    brief-selected-info 是否将选中项显示成简略信息 （只显示已选中xx个，可以通过selected-count-text设置对应的文案） （单选模式下此参数无效） boolean — false
    popper-class 为 treeselect 的 popper 添加类名 — -
    clearable 检索栏 输入框可清空 boolean — true
    item-height 每个节点的高度 number — 34
    search-height 检索栏高度 number — 44
    lazy 是否懒加载子节点，需与 load 事件结合使用 boolean — false
    select-clearable 选中数据是否可清除 boolean — false
    search-leaf-model 检索叶子节点模式 检索的时候，只检索非noStatistics的叶子节点，并以列表形式呈现选择结果，注意：此模式下，点击全选时候会触发 check-change事件，但是参数为空 boolean — false

    # 方法

    方法名 说明 参数
    filter 对树的 label 进行查找显示 (searchText) 用于查找的文本
    getCurrentKey 获取当前被选中节点的 id，若没有节点被选中则返回 null -
    setCurrentKey 通过 key 设置某个节点的当前选中状态 (key) 待被选节点的 id，若为 null 则取消当前高亮的节点
    getCurrentNode 获取当前被选中节点的 data，若没有节点被选中则返回 null -
    getCheckedNodes 若节点可被选择（即 show-checkbox 为 true），则返回目前被选中的节点所组成的数组 (filter方法, leafOnly) filter 会传入选中节点对象， 会根据返回值判断是否返回当前节点, leafOnly只返回子节点的
    getCheckedKeys 若节点可被选择（即 show-checkbox 为 true），则返回目前被选中的节点的 id 所组成的数组 (filter方法, leafOnly) filter 会传入选中节点对象， 会根据返回值判断是否返回当前id, leafOnly只返回子节点的 id
    getHalfCheckedNodes 若节点可被选择（即 show-checkbox 为 true），则返回目前被半选中的节点所组成的数组 (filter方法) filter 会传入选中节点对象， 会根据返回值判断是否返回当前节点
    getHalfCheckedKeys 若节点可被选择（即 show-checkbox 为 true），则返回目前被半选中的节点的 id 所组成的数组 (filter方法) filter 会传入选中节点对象， 会根据返回值判断是否返回当前id
    setCheckedKeys 通过 ids 设置目前勾选的节点 ( ids)，要勾选的id数组，如果想取消全部选中，可传[]， 全选传"ALL"
    setCheckedNodes 通过 node 设置目前勾选的节点 ( nodes)，要勾选的节点数组，如果想取消全部选中，可传[]， 全选传"ALL"
    setChecked 根据 key 设置某个节点的 选中状态 (key, value) （id, true或false）
    setNodeAttributeByKey 根据 key 设置某个节点的 某个属性 的值 (key, name, value) （id, 属性名，属性值）
    getNode 通过 id 获取节点 -
    setChildrenChecked 根据 parentKey 设置所有子节点的 选中状态（对于部门选择的变态关联十分有效） (parentKey, value) （id, true或false）
    setPopoverVisibility 设置popover 浮层显示状态 (visible) （true或false）
    getSelectedNodes 返回 实际显示的 被选中的节点所组成的数组， 单选模式下，也返回数组 -
    getSelectedKeys 返回 实际显示的 被选中的节点key所组成的数组， 单选模式下，也返回数组 -

    # item 属性 (入参data的子项)

    参数 说明 类型 默认值
    disabled 禁用某个节点的复选框 boolean false
    noStatistics 如果某个叶子节点需要不纳入统计，可设置此字段为 true，将不会出现在左侧选中数据中 boolean false

    # props (Attributes中的 props 参数的子项)

    参数 说明 类型 默认值
    disabled 禁用某个节点的复选框，此优先级高于item数据中的 disabled function(node, data){ return boolean } -
    children 子节点字段 string 'children'
    label 节点的显示文字字段 string 'label'

    # 事件

    事件名 说明 参数
    node-click 节点被点击时的回调 共两个参数，依次为：当前节点对象、节点的源数据
    check-change 节点选中状态发生变化时的回调 共两个参数，依次为：当前节点对象、节点本身是否被选中，【如果是清除选择触发的check-change，无参数】
    node-drop 拖拽放手时触发的事件(不会真的移动节点，只是推出事件，具体数据层次操作需外部手动完成), 如果是树以外的地方接收，通过e.dataTransfer.getData('draggingData')可获取到拖拽的节点的源数据(已转化为 String 类型)) 共四个参数，依次为：被拖拽节点对应的源数据、结束拖拽时最后进入的节点的源数据、被拖拽节点的放置位置（before、after、inner）、event
    tree-update-ready 树节点渲染完成触发 一个参数：全部的节点数据
    node-click 节点被点击时的回调 共两个参数，依次为：当前节点对象、节点的源数据
    single-check-change 单选模式时节点选中状态发生变化时的回调 共两个参数，依次为：当前节点对象、节点本身是否被选中
    node-drag-start 拖拽开始 共两个参数，依次为：被拖拽节点、event
    node-drag-end 拖拽结束 共两个参数，依次为：被拖拽节点、event
    popover-visible-change 浮层显示状态改变 共一个参数：是否显示浮层（true/false）
    load 加载子树数据的方法，仅当 lazy 属性为true 时生效 共两个参数，依次为：node节点对象、resolve()回调，回调函数的参数为新子节点数组
    after-filter 当树发生筛选查找完成触发 nodes 筛选之后显示的节点
    clear 清除选中时触发, select-clearable为true时，点击清除icon，会触发此事件 -
    filtered-nodes-all-check-change search-leaf-model为true时，点击全选时触发 两个参数，依次为：nodes 筛选之后的数据中选中状态发生改变的节点， checked 是否选中

    # Scoped Slot

    name 说明
    - 自定义树节点的内容，参数为 { node, data , text} {节点，节点源数据，检索框的值}
    icons 自定义树节点图标，参数为 { node, data} {节点，节点源数据}
    selected 自定义左侧选中栏内部的显示，参数为 { count, items} {选中节点个数， 选中的节点}
    empty 用于替换 "暂无数据"
    search 用于替换 "检索栏"
