    # Button 按钮

    常用的操作按钮。

    # 基础用法

    基础的按钮用法。

    使用type、plain、round和circle属性来定义 Button 的样式。

    <meg-row>
  <meg-button>默认按钮</meg-button>
  <meg-button type="primary">主要按钮</meg-button>
  <meg-button type="success">成功按钮</meg-button>
  <meg-button type="info">信息按钮</meg-button>
  <meg-button type="warning">警告按钮</meg-button>
  <meg-button type="danger">危险按钮</meg-button>
</meg-row>

<meg-row>
  <meg-button plain>朴素按钮</meg-button>
  <meg-button type="primary" plain>主要按钮</meg-button>
  <meg-button type="success" plain>成功按钮</meg-button>
  <meg-button type="info" plain>信息按钮</meg-button>
  <meg-button type="warning" plain>警告按钮</meg-button>
  <meg-button type="danger" plain>危险按钮</meg-button>
</meg-row>

<meg-row>
  <meg-button round>圆角按钮</meg-button>
  <meg-button type="primary" round>主要按钮</meg-button>
  <meg-button type="success" round>成功按钮</meg-button>
  <meg-button type="info" round>信息按钮</meg-button>
  <meg-button type="warning" round>警告按钮</meg-button>
  <meg-button type="danger" round>危险按钮</meg-button>
</meg-row>

<meg-row>
  <meg-button icon="meg-icon-search_1" circle></meg-button>
  <meg-button type="primary" icon="meg-icon-edit_photo" circle></meg-button>
  <meg-button type="success" icon="meg-icon-confirm" circle></meg-button>
  <meg-button type="danger" icon="meg-icon-delete_1" circle></meg-button>
</meg-row>

<style scoped>
  .meg-row{
    padding-bottom: 20px;
  }

    # 禁用状态

    按钮不可用状态。

    你可以使用disabled属性来定义按钮是否可用，它接受一个Boolean值。

    <meg-row>
  <meg-button disabled>默认按钮</meg-button>
  <meg-button type="primary" disabled>主要按钮</meg-button>
  <meg-button type="success" disabled>成功按钮</meg-button>
  <meg-button type="info" disabled>信息按钮</meg-button>
  <meg-button type="warning" disabled>警告按钮</meg-button>
  <meg-button type="danger" disabled>危险按钮</meg-button>
</meg-row>

<meg-row>
  <meg-button plain disabled>朴素按钮</meg-button>
  <meg-button type="primary" plain disabled>主要按钮</meg-button>
  <meg-button type="success" plain disabled>成功按钮</meg-button>
  <meg-button type="info" plain disabled>信息按钮</meg-button>
  <meg-button type="warning" plain disabled>警告按钮</meg-button>
  <meg-button type="danger" plain disabled>危险按钮</meg-button>
</meg-row>

    # 文字按钮

    没有边框和背景色的按钮。

    <meg-button type="text">文字按钮</meg-button>
<meg-button type="text" disabled>文字按钮</meg-button>
    show code

    # 图标按钮

    带图标的按钮可增强辨识度（有文字）或节省空间（无文字）。

    设置icon属性即可，icon 的列表可以参考 icon 组件，也可以设置在文字右边的 icon ，只要使用i标签即可，可以使用自定义图标。

    <meg-button type="primary" icon="meg-icon-search_1">搜索</meg-button>
<meg-button type="primary" icon="meg-icon-edit_photo">编辑</meg-button>
<meg-button type="primary" icon="meg-icon-confirm">确认</meg-button>
<meg-button type="primary">上传<meg-icon name="meg-icon-upload" class="meg-icon--right"></meg-icon></meg-button>
    show code

    # 按钮组

    以按钮组的方式出现，常用于多项类似操作。

    使用<meg-button-group>标签来嵌套你的按钮。

    <meg-button-group>
  <meg-button type="primary" icon="meg-icon-arrow_left">上一页</meg-button>
  <meg-button type="primary">下一页<meg-icon name="meg-icon-arrow_right" class="meg-icon--right"></meg-icon></meg-button>
</meg-button-group>
<meg-button-group>
  <meg-button type="primary" icon="meg-icon-edit_photo"></meg-button>
  <meg-button type="primary" icon="meg-icon-delete_1"></meg-button>
</meg-button-group>
    show code

    # 加载中

    点击按钮后进行数据加载操作，在按钮上显示加载状态。

    要设置为 loading 状态，只要设置loading属性为true即可。

    <meg-button type="primary" :loading="true">加载中</meg-button>
    show code

    # 不同尺寸

    Button 组件提供除了默认值以外的三种尺寸，可以在不同场景下选择合适的按钮尺寸。

    额外的尺寸：medium、small、mini，通过设置size属性来配置它们。

    <meg-row>
  <meg-button>默认按钮</meg-button>
  <meg-button size="medium">中等按钮</meg-button>
  <meg-button size="small">小型按钮</meg-button>
  <meg-button size="mini">超小按钮</meg-button>
</meg-row>
<meg-row>
  <meg-button round>默认按钮</meg-button>
  <meg-button size="medium" round>中等按钮</meg-button>
  <meg-button size="small" round>小型按钮</meg-button>
  <meg-button size="mini" round>超小按钮</meg-button>
</meg-row>

    # Attributes

    参数 说明 类型 可选值 默认值
    size 尺寸 string medium / small / mini —
    type 类型 string primary / success / warning / danger / info / text —
    plain 是否朴素按钮 boolean — false
    round 是否圆角按钮 boolean — false
    circle 是否圆形按钮 boolean — false
    loading 是否加载中状态 boolean — false
    disabled 是否禁用状态 boolean — false
    icon 图标类名 string — —
    autofocus 是否默认聚焦 boolean — false
    native-type 原生 type 属性 string button / submit / reset button
